Input Stack!

An online singleplayer or multiplayer game of crazy proportions. Go in alone, or with friends, and face an infinitely growing list of modules. Modules slowly spawn in, needing more and more work and teamwork.

This is made possible using google's Firebase Realtime Database to keep browsers in-sync of game data such as game time, module type, input values, player names, and more necessary information. Probably my most favorite database project I have made so far.

Writing Assault V2

Answer prompts and try to find the odd one out. Can you find the odd one out?

Version one was released around 2018-2019 as my first attempt at using Firebase's database. It was a little messy, so Version 2 (the version here) improved many issues to keep the game playable. It was a noble start into learning how to utilize firebase that taught me a lot of things about the capability of multiplayer games and synced sites.

Paint!

An online pixel art painting site. All changes to the canvas updates between users in real time. Also, save screenshots of the current canvas to save the current canvas to keep your art stored without worry.
Uses google's Firebase Realtime Database, a cloud-hosted database, to store canvas information (painted pixels & colors).
